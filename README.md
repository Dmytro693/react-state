# React State Lab

Лабораторна робота з React на тему `useState` та керування станом форми.

## Реалізовано

- компонент `UserForm`
- використання `useState`
- збереження полів в одному об'єкті `formData`
- відображення введених даних під формою у вигляді списку

## Поля форми

- Ім'я
- Прізвище
- Код групи
- Електронна пошта

## Запуск проєкту

```bash
npm install
npm start
```
